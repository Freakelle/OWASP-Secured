About this demo

The original OWASP Juice Shop is an intentionally insecure web application developed by OWASP for security training and awareness. It covers vulnerabilities from the OWASP Top 10 and beyond including SQL injection, XSS, broken authentication, and dozens more. It's widely used in CTFs, security courses, and penetration testing practice because every flaw is real and exploitable.

This site is a replica that mirrors Juice Shop's core UI for product browsing, search, login, and registration but with the vulnerabilities being patched.
